# Siber & Technology Development — Recruitment Website

A futuristic, cyberpunk-themed recruitment and promotional website for **Biro Siber** and **Biro Pengembangan Teknologi** — two elite technology divisions of a technology organization.

## Tech Stack

- **Next.js 14** (App Router, Static Export)
- **TypeScript**
- **Tailwind CSS** — utility-first styling with custom cyber/neon theme
- **Framer Motion** — scroll reveal, tilt cards, staggered animations
- **Google Fonts** — Orbitron (display), Exo 2 (body), Share Tech Mono

## Running Locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Building for Production

```bash
npm run build
```

Output is generated in the `out/` directory (static export, ready for Netlify).

## Pages & Sections

1. **Hero** — Animated canvas grid, glitch text, CTA buttons, live clock in navbar
2. **About** — Glassmorphism panels describing both divisions
3. **Divisions** — Interactive tilt cards with capability bars
4. **Why Join** — 6-card asymmetric grid of benefits
5. **Gallery/Team** — Placeholder team grid with hover overlays
6. **CTA** — Full-screen dramatic closing section
7. **Footer** — Social links, navigation, system status
